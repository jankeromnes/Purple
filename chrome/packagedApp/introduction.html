Purple, a QueryPoint Debugger for Web Pages

Purple runs your Web application in an enhanced Web browser with special new powers(*):

  • Supports Powerful Proposed ECMAScript Language Features
  • Instant Live Updates from Source Integrated with Online Web Development
  • Next Generation QueryPoint Debugging for JavaScript

Let's Go!

  1. Enter the URL for your Web Site in the input box
  2. Click "Load Web Application"
  3. This window will convert into the debugger, and new window will open with your web app.
  4. Hopefully there will be more instructions here when I figure them out

(*) Ok some of these powers are still under development
